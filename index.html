How to use

Update prices in seconds

  1. Click Edit on any item and adjust the local price, serve size or macros.
  2. Use Export JSON to save your edited dataset.
  3. Commit the exported JSON over data/foods.json for all GitHub Pages visitors.
GitHub Pages is static, so shared public updates happen through the repo. Personal edits save instantly in your browser via localStorage.
